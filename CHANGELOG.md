# Changelog

## [1.0.0] - 2026-01-10

### Added
- Initial release of GitHub Streaks
- Menu bar streak counter showing current contribution streak
- Mini contribution grid in menu bar
- Full year contribution graph view in popover
- Customizable display options (streak only, grid only, or both)
- Settings panel for GitHub username configuration
- Launch at login option
- Native macOS design with dark/light mode support
- Automatic data refresh with caching
- Tooltip with contribution details on hover

### Technical
- Built with SwiftUI for macOS 14.0+
- Efficient HTML parsing for GitHub contribution data
- Local caching to minimize network requests
